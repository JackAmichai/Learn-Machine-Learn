## 2024-03-24 - Async dataSync \n**Learning:** The usage of `dataSync()` is synchronous and blocks the UI thread, causing jank. The solution is to use async `data()` instead, wrapping it inside async functions. Tensors initialized during async data access need manual `dispose` to prevent leaks.\n**Action:** Use `await tensor.data()` instead of `dataSync()` and manage tensor disposals manually if inside async logic.
