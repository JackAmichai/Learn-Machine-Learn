## 2025-02-14 - Refactoring tfjs synchronous hooks
**Learning:** Tensorflow.js' `dataSync()` blocks the main thread. It can easily be refactored to use `await tensor.data()` to speed things up, however we cannot use `tf.tidy()` to do cleanups since it's only synchronous. We must do cleanup in finally blocks.
**Action:** When migrating synchronous TF.js logic that reads weights/data to `async` context to prevent UI jank, ensure manual `tensor.dispose()` calls inside `finally` blocks, and make sure state setters use an `isMounted` ref flag.
