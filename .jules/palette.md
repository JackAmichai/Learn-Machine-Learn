## 2024-03-20 - Absolute Positioned Buttons Over Text
**Learning:** When adding absolute-positioned elements (like a "Copy" button) over scrollable text/code blocks, using semi-transparent backgrounds (e.g., `rgba()`) causes the underlying text to bleed through and clash visually with the button text when users scroll.
**Action:** Always use solid background colors for absolute-positioned UI elements that sit on top of content to ensure optimal contrast and readability.
